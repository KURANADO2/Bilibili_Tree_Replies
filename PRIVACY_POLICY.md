# Privacy Policy

**Bilibili Tree Replies**

Effective date: 2026-04-30

This privacy policy explains how the Bilibili Tree Replies Chrome extension handles data.

## What the extension does

The extension runs on Bilibili video pages and displays comment replies in a tree view. To do that, it reads comment data from Bilibili and can send comment-related actions back to Bilibili when you choose to like, dislike, or reply.

## Data access and use

The extension may access:
- The current Bilibili video page URL
- Comment content, author names, avatars, timestamps, like counts, and reply structure visible on the page
- The Bilibili account session already present in your browser, only to perform comment actions you initiate

The extension uses this data only to:
- Load and render the tree view
- Load more comments and replies
- Submit likes, dislikes, and replies when you click those actions
- Switch between tree view and the original comment area

## Data storage

The extension does not maintain a separate user account.
It does not store comment data on an external server.
Any state it keeps is local to the extension and the current browser session.

## Data sharing

The extension does not sell user data.
The extension does not send data to third-party services other than Bilibili endpoints required for the extension to function.

## Permissions

The extension uses:
- `https://www.bilibili.com/*` to run on Bilibili video pages
- `https://api.bilibili.com/*` to load and update comment data

## Contact

If you have questions about this extension, contact the developer through the project page or store listing.
